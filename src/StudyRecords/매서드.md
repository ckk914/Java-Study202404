- 자바의 함수는
- 반드시 클래스 블록 내부,
-  다른 함수 외부에 만들어야함
- 함수를 호출할 때는 함수 안에서만 가능
- void 함수는 함수의 결과를 변수에 담을 수 없음.
- 반드시 단독 호출해서 사용


-함수 정의 위치는 메인 위아래 상관없음
